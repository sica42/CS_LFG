# CS LFG
**Version:** 0.4
**Game Version:** Crusader Storm WoW (2.4.3 and 2.5.3 clients)

LFG Interface for Crusader Storm LFG commands. No automated group matching yet, so you'll need to form your own groups.

## Features

* Add yourself or you group to the queue via an interface.
* Adding group to queue supports role confirmation for each player if they also have addon installed.
* Browse LFG list. Can view all entries, grouped by dungeons or only groups looking for more members.
  * When viewing all, you can double click an entry to whisper the player.

## Installation

1. Download the [latest release](https://github.com/sica42/CS_LFG/releases/latest/download/CS_LFG.zip) or clone the addon into your `Interface\AddOns\` folder.
2. Make sure the folder is named `CS_LFG`.
3. Restart WoW or do a `/reload`.
